# Changelog

## 0.1.0-alpha.0

- Added 47 cross-platform components, design tokens, and a full Playground catalog.
- Added focused logic dependencies for dates, validation, decimal precision, and events.
- Added light/dark themes, semantic states, Overlay Host, media fallback states, and a 64-icon Lucide subset.
- Added bilingual documentation and component runtime locale support.

## Version Policy

- Breaking API changes use a major version.
- Backward-compatible optional capabilities use a minor version.
- Fixes use a patch version.
