# 更新日志

## 0.1.0-alpha.0

- 建立 47 个跨端组件、主题 Token 与完整 Playground 目录。
- 引入 Day.js、async-validator、decimal.js-light 与 mitt 作为纯逻辑依赖。
- 提供亮暗主题、语义状态、Overlay Host、媒体失败回退与 Lucide 64 图标子集。
- 新增正式文档站：组件 API、代码示例、移动端预览和本地搜索。

## 版本策略

- 破坏性 API 调整使用主版本升级。
- 新增可选 Props 与不改变默认行为的能力使用次版本升级。
- 修复交互、样式和文档错误使用补丁版本升级。
- 废弃能力至少保留一个次版本，并在迁移文档中给出替代方案。
